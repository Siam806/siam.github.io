    Developing for the web

  • Brackets
    Because I started to learn HTML as my first language I had to find a good editor wich I would stick to throughout becoming more skilled in the language but it also had to be beginner friendly. I tried out Atom, Sublime, Notepad++ and Kate but by far the easiest was Brackets. Not only did it allow me to install awesome extensions like "Beautify" which sorts the text for you so you can concentrate on the writing and don't have to worry about people not able to understand what you have written. The UI is very simple but has lots of features. Not only does it handle HTML but also of course CSS & javascript. To see the list of extensions I have installed click here.
